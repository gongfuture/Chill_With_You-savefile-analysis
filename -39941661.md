### 环境音设置

#### 字典部分示例

```json
"AmbientSoundDic" : {0:{
     "AmbientSoundType" : 0,
     "SoundVolume" : 0.5,
     "IsMuteAmbient" : true
    },1:{
     "AmbientSoundType" : 1,
     "SoundVolume" : 0.5,
     "IsMuteAmbient" : false
    }
}
```

##### 字段说明

- 序号：环境音id
- AmbientSoundType：环境音id，应与序号一致

##### 一些小说明

*: 关于烟花5、6，怀疑6是目前没用上的占位，或者废案。6声音会被重置为0且默认静音，具体可见下面测试

<details>
<summary>关于烟花5、6的详细测试</summary>

1. 5、6都不打开
    如同正常不开烟花的情况
    此时游戏内打开烟花，存档变成如下状态

    ```json
    5:{
        "AmbientSoundType" : 5,
        "SoundVolume" : 0.5,
         "IsMuteAmbient" : false
    },6:{
         "AmbientSoundType" : 6,
         "SoundVolume" : 0,
         "IsMuteAmbient" : true
    }
    ```

2. 只打开5
    如游戏内正常打开烟花的样子，不多赘述

3. 只打开6
    无效果，没有烟花声音，UI也不显示激活

4. 将6的声音设置为0.5
    开启游戏后声音被重置为0，开关不变

5. 5、6都开，且声音都为0.5
    如同2情况，但是存档状态为

    ```json
    5:{
        "AmbientSoundType" : 5,
        "SoundVolume" : 0.5,
         "IsMuteAmbient" : false
    },6:{   
         "AmbientSoundType" : 6,
         "SoundVolume" : 0.5,
         "IsMuteAmbient" : false
    }
    ```

</details>
